Campaign Widget Service - Demo

This page demonstrates different ways to embed campaign widgets using the Campaign Widget Service.

1. Basic Widget (Medium Size, Light Theme)

The simplest implementation with default settings.

HTML:
<iframe src="http://localhost:3002/widget?campaignName=demo-campaign&userId=user-456" width="350" height="250" frameborder="0" style="border-radius: 12px;"></iframe>

2. Small Widget (Top-Right Position)

Smaller widget perfect for corner placement.

HTML:
<iframe src="http://localhost:3002/widget?campaignId=demo-456&userId=user-789&size=small&position=top-right" width="200" height="150" frameborder="0"></iframe>

3. Large Widget (Dark Theme)

Large widget with dark theme for better visibility on light backgrounds.

HTML:
<iframe src="http://localhost:3002/widget?campaignName=demo-large&userId=user-123&size=large&theme=dark" width="450" height="320" frameborder="0" style="border-radius: 12px;"></iframe>

4. Dialogue Widget (Backend HTML)

Dialogue widget that displays custom HTML content from the backend recommendation API.

HTML:
<iframe src="http://localhost:3002/widget?campaignName=dialogue-demo&userId=user-456&contentType=dialogue&size=dialogue" width="400" height="350" frameborder="0" style="border-radius: 12px; resize: both; overflow: auto;"></iframe>

5. Modal Widget (Full Backend Control)

Modal widget with larger dimensions for complex recommendation content.

HTML:
<iframe src="http://localhost:3002/widget?campaignName=modal-demo&userId=user-789&contentType=modal&size=modal&objective=purchase" width="500" height="400" frameborder="0" style="border-radius: 12px; box-shadow: 0 8px 32px rgba(0,0,0,0.12);"></iframe>

4. Dynamic Widget Loading with JavaScript

Load widgets dynamically based on user behavior or conditions.

JavaScript:
function loadWidget(containerId, config) {
  const iframe = document.createElement('iframe');
  const params = new URLSearchParams(config);
  iframe.src = `http://localhost:3002/widget?${params.toString()}`;
  iframe.width = config.width || '300';
  iframe.height = config.height || '200';
  iframe.frameBorder = '0';
  document.getElementById(containerId).appendChild(iframe);
}

Widget Configuration Options

Parameter Options Description
campaignName Any string Campaign name for backend API lookup
userId Any string User identifier for personalization
size small, medium, large, dialogue, modal Widget dimensions and layout type
contentType widget, dialogue, modal Content presentation style
theme light, dark Visual theme
objective Any string User objective for recommendation targeting
brandName Any string Brand context for recommendations
position top-left, top-right, bottom-left, bottom-right, center Widget placement (for overlay widgets)
